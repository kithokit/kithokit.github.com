Ruby ||= Operator

This topic is talked about ruby ||= and || operator It is always confused with ruby newbie

||= will set the left-hand value to the right hand value only if the left-hand value is falsey OR undefined.

Example 1, if a is defined as 4, using a ||= 5 will still output 4
